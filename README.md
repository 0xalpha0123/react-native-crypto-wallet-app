# Zluck React Native
```
1) git clone https://gitlab.com/zluck/mobile/react-native/zluck-rn.git <ProjectName>

2) cd <ProjectName>

3) yarn

5) yarn global add react-native-rename
                  or
  npm install react-native-rename -g

6) npx react-native-rename <ProjectName> -b <bundleIdentifier>

7) react-native link

8) cd ios && pod install && cd ..
```
